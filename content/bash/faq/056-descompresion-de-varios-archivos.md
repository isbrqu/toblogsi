title = '56. ¿Cómo puedo descomprimir varios archivos comprimidos a la vez?'
date = 2023-09-30T13:55:54-03:00
draft = false
+++

Como el comando tar se diseñó originalmente para leer y escribir en
dispositivos de cinta (tar - Tape ARchiver), puede especificar solo
nombres de archivos para colocarlos dentro de un archivo (escribir en
cinta) o extraerlos de un archivo (leer desde cinta).
